Introduction


In this module we will look at how exploratory data analysis is performed using pandas by building and manipulating DataFrames. Following on from this we will explore visualisations using matplotlib and seaborn, as well as investigate the basics of hypothesis testing.



Learning Objectives


  • Build and manipulate DataFrames using Pandas
  • Explore visualisations suites in python to present data in various ways
  • Perform a hypothesis test using the stats library
