Tools of the Trade



Introduction


In this module we will be looking at different tools you may use in your role, including Big Data technologies and Data Analytics Platforms. We will consider the advantages of using such approaches and contrast them with the oding techniques you have learned over the previous sessions.

In the second half of the module we will explore R, a language built for statistical analysis. We will see how techniques covered in python, including data cleaning, visualisations and basic modelling can be performed in R.



Learning Objectives


  • Understand fundamental concepts of Big Data
  • Justify the use (or lack of) Big Data technologies in your analysis
  • Critically evaluate the differences in using a Data Platform and Coding it Yourself
  • Understand how R can be utilised in your analysis
