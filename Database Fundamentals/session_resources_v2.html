Database Fundamentals



Introduction


In the second Multiverse module we will learn about two important databases: normalised relational databases and NoSQLto understand how and why they are used for analysis.

Following on from this we will consider how concepts such as data integration and data warehouses can be used to enhance our analysis..



Learning Objectives


  • Explain the concepts and uses of a relational database management system
  • Identify the different types of key in a RDBMS
  • Understand the principles of normalisation on a relational database
  • Design and visualise an entity relationship diagram for a database
  • Understand the principles of data integration and how it is used to help our analysis
  • Describe the ETL process and how it is used to in data integration
